# Housing Price Intelligence

Aplikasi Flask: **Home**, **Dataset**, **Regresi**, **Klasifikasi**, **Clustering**, **Prediction**, **About** + model dari Kaggle.


## Halaman aplikasi

| URL | Isi |
|-----|-----|
| `/` | Landing page |
| `/dataset` | KPI & grafik interaktif EDA |
| `/regression` | Metrik regresi, SHAP, prediksi harga |
| `/classification` | Metrik klasifikasi, SHAP, prediksi High/Low |
| `/clustering` | Silhouette, scatter cluster, profil cluster |
| `/prediction` | 3 tab form + hasil + penjelasan fitur + SHAP |
| `/predict` | Redirect ke `/prediction` |
| `/about` | Info proyek & tim |
